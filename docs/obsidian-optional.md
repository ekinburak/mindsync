# Obsidian (Optional)

The LLM Wiki is just a folder of markdown files. Any editor works. Obsidian adds useful features for browsing and navigating your wiki.

## Why Obsidian (if you want it)

- **Graph view** — see how pages link to each other visually
- **Backlinks panel** — see everything that links to the current page
- **Web Clipper** — browser extension that converts web articles to markdown for `raw/`
- **Dataview plugin** — query your wiki by frontmatter tags and dates

## Setup tips

1. Open Obsidian → Open folder as vault → select your wiki folder
2. Settings → Files and links → set Attachment folder to `raw/assets/`
3. Install Web Clipper from the Chrome Web Store
4. Bind "Download attachments" to a hotkey (Ctrl+Shift+D) to save images locally

## You don't need Obsidian

VS Code, Typora, iA Writer, or plain Terminal all work. The wiki is markdown — editor is your choice.
